# Nexcent

A modern, responsive landing page built with HTML and CSS.

## Overview

Nexcent is a clean and professional landing page featuring multiple sections including hero, features, testimonials, FAQ, and footer. The design is fully responsive and optimized for all device sizes.

## Technologies Used

- **HTML5** - Semantic markup structure
- **CSS3** - Styling, animations, and responsive design
- **Font Awesome** - Icon library for social media and UI icons
- **Google Fonts** - Inter font family

## Features

- Fully responsive design (mobile, tablet, desktop)
- Smooth scroll navigation
- Interactive hover effects
- Modern UI components
- Clean and professional design
- Mobile-friendly navigation menu

## Sections

- **Hero Section** - Main landing area with call-to-action
- **Our Clients** - Client logos slider
- **Features** - Service features with cards
- **Content Sections** - Information sections with images
- **Stats Section** - Statistics display
- **Testimonial Section** - Customer testimonials
- **FAQ Section** - Frequently asked questions
- **CTA Section** - Call-to-action section
- **Footer** - Footer with links and newsletter subscription
